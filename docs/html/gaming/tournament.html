Модуль Tournament¶

class gaming.tournament.Tournament¶

Базовые классы: object

Класс для управления турнирами и соревнованиями.

Обеспечивает функционал создания турниров, регистрации игроков, ведения рейтинга и распределения призового фонда.

Methods:

create_schedule(start_date, end_date): Создание расписания турнира register_player(player_id, initial_rating): Регистрация участника calculate_prize_distribution(): Расчет призовых мест update_leaderboard(player_id, result): Обновление таблицы лидеров get_tournament_rules(): Получение правил турнира start_tournament(): Запуск турнира

Attributes:

players (list): Список участников турнира prize_pool (float): Призовой фонд status (str): Статус турнира min_players (int): Минимальное количество участников rating_points (dict): Рейтинговые очки игроков schedule (list): Расписание турнира rules (dict): Правила проведения leaderboard (dict): Таблица лидеров
