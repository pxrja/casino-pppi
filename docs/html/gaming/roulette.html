Модуль Roulette¶

class gaming.roulette.Roulette(variant='European')¶

Базовые классы: object

Класс для реализации игры в рулетку с разными вариантами правил.

Поддерживает европейский, американский и французский варианты, различные типы ставок и систему выплат.

Methods:
