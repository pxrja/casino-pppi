Модуль Poker¶

class gaming.poker.Poker(variant='Texas Holdem')¶

Базовые классы: object

Класс для реализации различных вариантов покера.

Поддерживает Texas Holdem, Draw Poker и Omaha, включая кэш-игры и турнирный режим.

Methods:
