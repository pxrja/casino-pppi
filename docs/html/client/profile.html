Модуль Profile¶

class client.profile.Profile¶

Базовые классы: object

Класс для управления профилем пользователя игровой платформы.

Обеспечивает функционал регистрации, настройки профиля, отслеживания активности и статистики игрока.

Methods:

register(username, email, password): Регистрация нового пользователя update_settings(setting_name, value): Обновление настроек профиля add_activity(activity): Добавление записи активности upgrade_vip_status(new_status): Повышение VIP статуса update_tournament_rating(points): Обновление турнирного рейтинга get_stats(): Получение статистики игрока calculate_win_rate(): Расчет процента побед
